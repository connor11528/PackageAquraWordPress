Getting Started

Download

After you purchased AQURA from Themeforest you have to go to Downloads panel and download the .zip archive containing the WordPress installation file.

If you read this documentation, you probably downloaded the archive already.

Installation

There are 2 ways to install the theme:

  1. with WordPress theme installer;
  2. by FTP;

  1. With WordPress theme installer

    Go to Appearance->Themes from your WP Dashboard and press Add New button.

    Then go to Upload tab, browse for aqura.zip archive, select it and hit Install Now. After install is finished, activate the theme.

    After you finish the activation of the theme, you will be noticed to install some plugins. Hit Begin installing plugins and install & activate all of them.

    That's it! Now you have the theme installed.


  2. by FTP

    With FTP, the only difference is the uploading system. You have to access with some FTP client your server and the installation of the WP. Then go to wp-content/themes/ and upload the .zip theme. After you finished the uploading, go and unzip it.

    After you complete this, you have to go to Appearances->Themes and activate the theme and its plugins.

After installation theme and all required plugins you can import the demo content by going to Tools > Import > WordPress and upload "aqura-demo-content.xml"

Post Options

With post options you are able to:

  1. Manage breadcrumb title on "Top Section"
  2. Manage breadcrumb description on "Top Section"
  3. Chose the article type on "Article Type"
  4. Chose video ID (Vimeo/Youtube) or SoundCloud ID on "Article Type"
  5. Set gallery images on "Article Type"

Page Options

With page options you are able to:

  1. Manage breadcrumb title
  2. Manage breadcrumb description
  3. Manage number of posts per page on listing templates
  4. Chose form title on contact template
  5. Set background image for contact section

We have activate the Visual Composer to make creation of a page more easy and customizable.

Gallery Options

With gallery options you are able to:

  1. Manage the gallery title. If you want to make bold words you must put the words into a span tag, e.g: "<span>Title</span>"
  2. Set the gallery photos
  3. Set the gallery thumbnail

Albums Options

With albums options you are able to:

  1. Set the page background
  2. Set the album cover
  3. Set the album disk image
  4. Manage the album details, e.g: Release Date, Format etc.
  5. Manage the album download links
  6. Set album songs

WPML Support

AQURA offers support for WPML plugin. Be aware that this plugin is not provided by us, and you will have to buy it in order to use it : WPML Official.

  • Using .mo/.po files

    • Our theme provide .mo/.po language files in the languages folder of the theme and you can use these files on WPML plugin. To use translation from a .mo file, choose the first option from the theme localization menu. The only thing that you need to specify is the text-domain that is 'aqura' for our theme. You need to enter this value in the text domain field, so that WPML can load the translation file and assign it to the theme.

  • Translating the theme from within WPML

    • When WPML localization is selected (the 2nd option in the image above), WPML will scan the theme and find the texts that can be translated. It reports what it finds and let you enter translations, right from within WordPress.

    Click on Scan the theme for strings and WPML will go through all the files in the active theme and get translatable texts from them (texts that are wrapped with gettext functions).



    It shows how many strings were found in the theme and their translation status. To translate them, click on View strings that need translation. This will take you to the String translation page, where you can enter translation for each of the texts found in the theme.

2.1. Press "Edit Row" from the section you want to have background image.

</p>
